# 로그인 개요

- 겜만중 계정으로 로그인할 수 있다.
- **이메일 인증을 완료해야 로그인 가능**하다.
- 일반 로그인의 아이디/비밀번호 외에 **소셜 로그인(Naver, Google, Kakao)** 를 지원한다.

---

## 로그인 방법

### 일반 로그인
1. 로그인 페이지 접속: `http://localhost:3000/login`
2. 아이디(이메일)와 비밀번호 입력
3. 로그인 버튼 클릭 → 성공 시 메인 페이지로 이동

### 소셜 로그인
1. 로그인 페이지에서 소셜 로그인 선택 (Naver / Google / Kakao)
2. 각 플랫폼의 동의 절차 완료
3. 최초 연동 시 계정이 자동 생성/연결될 수 있다

---

## 이메일 인증 조건

- 회원가입 후 전송된 **인증 메일 확인 → 인증 완료**해야 로그인 가능
- 인증 메일이 오지 않으면 **스팸함 확인** 후 다시 요청
- 인증 전 로그인 시도 시 “이메일 인증 필요” 안내가 표시된다

---

## 비밀번호 재설정

- 로그인 화면의 **“비밀번호 찾기/재설정”** 링크 이용
- 등록 이메일로 인증 링크 전송 → 새 비밀번호 설정
- 재설정에도 **이메일 인증 절차**가 필요하다

---

## 오류/예외 처리

| 상황 | 메시지/안내 | 조치 |
|------|-------------|------|
| 이메일 미인증 | “이메일 인증이 필요합니다.” | 인증 메일 재발송 후 인증 |
| 비밀번호 불일치 | “아이디 또는 비밀번호가 올바르지 않습니다.” | 비밀번호 재설정 안내 |
| 소셜 동의 취소 | “소셜 로그인 동의가 취소되었습니다.” | 다시 시도 또는 일반 로그인 |
| 계정 없음 | “가입된 계정을 찾을 수 없습니다.” | 회원가입 안내 |
| 다중 시도 실패 | “잠시 후 다시 시도해주세요.” | 일정 시간 후 재시도 |

## 보안 권장

- 비밀번호는 8자 이상 / 영문 대소문자·숫자 조합 권장
- 공용 PC 사용 시 **자동 로그인 해제** 및 **로그아웃** 필수
- 장시간 미사용 시 세션이 만료될 수 있다

## FAQ

**Q. 로그인하려는데 이메일 인증이 필요하다고 나와요.**  
A. 가입 시 전송된 인증 메일로 인증을 완료한 뒤 다시 로그인하세요. 스팸함도 확인해주세요.

**Q. 비밀번호를 잊었어요.**  
A. 로그인 페이지의 **비밀번호 재설정**을 통해 이메일 인증 후 새 비밀번호를 설정하세요.

**Q. 소셜 로그인 계정을 일반 계정으로 바꿀 수 있나요?**  
A. 현재는 동일 이메일 기준으로 연동하여 사용합니다. 자세한 변경은 고객센터로 문의해주세요.
